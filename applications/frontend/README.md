# Frontend Service

## Overview
This is the application frontend microservice skeleton. It represents the client interface of our e-commerce demo application (not to be confused with the main DevOps Nexus dashboard frontend).

## Structure
* `src/`: Client application code.
* `tests/`: Basic validation test stubs.
* `Dockerfile`: Container image configuration.

## Development Setup
Install dependencies:
```bash
npm install
```

Run dev server:
```bash
npm run dev
```
