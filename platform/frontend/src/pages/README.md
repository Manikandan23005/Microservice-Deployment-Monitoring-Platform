# Portal Pages

## Purpose
Top-level layout view pages rendered in standard application navigation paths.

## Planned Views
* `Overview.tsx`: Landing dashboard summary representing system status.
* `Deployments.tsx`: Interface mapping service release tags and promotions.
* `Pods.tsx`: Real-time list of pod states and events.
* `Metrics.tsx`: Full-screen graphs showing resource limits and usages.
* `Logs.tsx`: Inbound logs console with search and namespace filters.
* `AICommand.tsx`: Interactive chat assisting developers with troubleshooting warnings.
