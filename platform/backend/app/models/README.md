# Pydantic Schema Models

## Purpose
Contains static type definitions and request/response validation schemas for FastAPI routes.

## Planned Modules
* `deployment.py`: Schema models mapping trigger/rollback data payloads.
* `pod.py`: Schema models representing serialized Kubernetes pod resources.
* `telemetry.py`: Data models mapping time-series metrics.
* `ai.py`: Schema structures for diagnostic triage payloads.
