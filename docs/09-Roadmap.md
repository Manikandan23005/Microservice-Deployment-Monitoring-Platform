# Detailed Release Roadmap

## Overview
This document details the development milestones and release cycles for the DevOps Nexus platform.

## Goals
- Define testing boundaries and exit criteria for version releases.
- Re-align phase milestones to reflect the core unified platform model.

## Implementation Plan
1. **Phase 1: Project Skeleton (v0.1.0) - Done:**
   - Folder structure initialization, requirements mapping, basic YAML stubs.
2. **Phase 2: Unified Platform Restructuring (v0.2.0) - Current Phase:**
   - Promoted dashboard directory to platform, defined FastAPI architecture backend, integrated Poetry configs and Docker Compose setups.
3. **Phase 3: Core API Services Integration (v0.3.0):**
   - Connect FastAPI to active Kubernetes cluster client API and observability tools.
4. **Phase 4: TypeScript React Client UI (v0.4.0):**
   - Develop dashboard grids, logs aggregates panels, and chart visualizations.
5. **Phase 5: Pluggable AI Assistant (v0.5.0+):**
   - Connect platform backends to local Ollama/OpenAI APIs.

## Future Work
* **Continuous Updates:** Update this roadmap file as feature sets are integrated.
* **Integrate Milestone Tracking:** Map GitHub issues directly to these roadmap targets.
